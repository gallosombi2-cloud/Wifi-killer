# 📡 WiFi-Killer-Scan v13.0

Suite de seguridad WiFi con interfaz gráfica (GUI) diseñada para pruebas de desautenticación profesional.

## ✨ Características
- **Bloqueo Universal:** Desconecta todos los equipos de una red.
- **Modo Samsung Killer:** Ataque dirigido de alta velocidad para dispositivos modernos con protecciones avanzadas.
- **Gestión de Monitor:** Activación y desactivación limpia de la tarjeta de red.

## 🛠️ Instalación en Kali Linux
1. `git clone https://github.com/TU_USUARIO/WiFi-Killer-Scan.git`
2. `cd WiFi-Killer-Scan`
3. `sudo bash setup.sh`
4. `sudo ./killer_scan.sh`
